# First semester exam project - The Community Science Museum

This project is my submission from the first semester exam project in the Noroff school of digital media and technology FED 1 programme.
My second coding project featuring only vanilla HTML and CSS.
This project is not applicable for professional use and should be viewed as a sandbox for my learning development in my first year studies.

The course assignment brief gave us CA files for images and logo for a fictional science museum, The Community Science Museum.
Our task was to take use of our skills learned in the courses in the first semester. Plan the project using project methodology and GitHub planning tools. Design project with low-fidelity wire-frames and create high-fidelity design and prototype with Figma. Then execute the project plan and create the design with vanilla HTML and CSS.

The design vision was to create a colorful tablet/app-like website for kids. Everything is extra large on purpose.

This website lacks full functionality with donation checkout flow. Each page should be viewed as a mock of what a real world application could look like.

## Features

- Index page with cards to different pages
- Visit us page with link to google maps
- Donation flow pages with donation form and confirmation
- Construction page for pages that were not completed in time
- Hamburger dropdown menu with use of max-width/auto overflow CSS trick
- Full responsive scaling across viewports from 320px VW to 1920px VW, with use of @media breakpoints
- Implementation of icons and fonts via FontAwesome free library scripting
